Empresas que oferecem serviços baseados no QGIS Desktop, QGIS Server, Web (QGIS web client e/ou Lizmap)

O grupo utilizadores QGIS PT pretende apresentar nesta página uma lista de empresas/freelancers Portuguesas/es que ofereçam serviços comerciais baseados no software QGIS. Para fazer parte desta lista é necessário fazer uma doação ao grupo utilizadores QGIS PT e respeitar algumas condições:

    Objectivo
  • Potenciar o mercado de serviços QGIS em Portugal;
  • Proporcionar às empresas um local para divulgar os seus produtos e serviços;
  • Incentivar as empresas a terem uma postura proativa e de suporte ao projecto QGIS;
  • Proporcionar aos utilizadores uma forma de conhecerem e escolherem empresas e seus serviços.

    • Se está interessado em acrescentar a sua empresa na lista que se segue, por favor nos contacte através do nosso endereço e-mail qgis.portugal@gmail.com para detalhes e de forma que o grupo utilizadores QGIS PT possa availar o seu caso.

Legenda da tabela:

  • QGIS Core Developer: empresa que é referenciada como "core contributor" na seguinte pagina http://qgis.org/en/site/forusers/commercial_support.html
  • Desenvolvimento Upstream/Bug fixing: empresa que contribui ao desenvolvimento upstream de QGIS. A contribuição (sob forma de código fonte) deve estar referenciada no repositório oficial do QGIS https://github.com/qgis/QGIS/commits/master
  • Desenvolvimento Extensões: empresa pode desenvolver extensões em Python para QGIS. Exemplos devem estar publicados no repositório oficial de extensões de QGIS https://plugins.qgis.org/plugins/?per_page=340
  • Desenvolvimento WebGIS: empresa que pode desenvolver aplicações WebGIS de código aberto (disponibilizadas com licença OSI compativel) baseadas nas tecnologias QGIS Server e nos frameworks QGIS web client, Lizmap ou outros (ex: Openlayers).
  • Assistência/Suporte: empresa que oferece serviços de Assistência/Suporte presencial ou remota sobre o uso e resolução de duvidas acerca QGIS Desktop ou Server.
  • Formação: empresa que organiza formações (ou participa nelas como entidade formadora) sobre QGIS Desktop, Server ou Web.
  • Alojamento: empresa que oferece serviços de alojamento (servidores dedicados ou partilhados) para aplicações WebGIS ou para serviços WMS/WFS/WCS baseados em QGIS Server/Web.
