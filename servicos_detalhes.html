Lista de empresas que oferecem serviços baseados no QGIS

Para fazer parte desta lista é necessário fazer uma doação ao grupo utilizadores QGIS PT e, no ultimo ano, ter respeitado pelo menos duas (2) das seguintes condições:

  • Ter contribuído ao código fonte do QGIS com novas funcionalidades ou bug fixes1
  • Ter publicado extensões (plugins) no repositório oficial de QGIS2
  • Ter disponibilizado para download alguma aplicação derivada do QGIS (Desktop, Server, Web) com licença OSI (Open Source Initiative) compatível http://opensource.org/licenses2
  • Ter contribuído ao projecto QGIS com uma doação em dinheiro, cujo valor mínimo depende do facturado da empresa3
  • Ter contribuído a tradução da documentação do QGIS ou da interface (GUI) do programa Desktop (só para freelancers)4

Para informações escreva para o nosso endereço e-mail qgis.portugal@gmail.com

1 É necessário o comprovar com os links as commits no repositório do codigo fonte do QGIS (https://github.com/qgis/QGIS/commits/master).
2 É necessário o comprovar com os links apropriados no repositório oficial de extensões de QGIS (https://plugins.qgis.org/plugins/?per_page=340) e comprovar a real utilizade da extensão. No caso de aplicações derivadas é necessário o comprovar com links para os respeitivos repositórios de codigo fonte (Github, Bitbucket, etc.).
3 No caso do freelancers são aceites pequenas doações.
4 É necessário o comprovar com os links as commits no repositório do codigo fonte do QGIS (https://github.com/qgis/QGIS/commits/master) ou no repositório da documentação (https://github.com/qgis/QGIS-Documentation).
